Display a breadcrumb on the page.

## Example

```html
<div style="padding: 20px">
    <div class="control-breadcrumb">
        <ul>
            <li><a href="#">Dash Board</a></li>
            <li><a href="#">Blog Posts</a></li>
            <li>Edit Post</li>
        </ul>
    </div>
</div>
```
